### Gram-Schmidt Method
```
'''Developed by D.Devadhaarini
Reg no.: 212223230040'''
num=eval(input())
for i in range(len(num)):
    low=i
    for j in range(i+1,len(num)):
        if num[j]<num[low]:
            low=j
    num[i],num[low]=num[low],num[i]
print(num)
```
